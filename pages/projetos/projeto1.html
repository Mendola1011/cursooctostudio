Projeto 1 - Pato (Introdutório)

Neste projeto introdutório, os alunos vão conhecer a interface do Octostudio e experimentar seus primeiros blocos de programação, desenvolvendo um pato animado se movimenta, pula e emite sons, tudo com poucos comandos. O objetivo principal é permitir que os estudantes se familiarizem com a lógica de programação visual, sem a pressão de ter que fazer um jogo complexo. O jogo do Pato é composto por apenas um ator e dois pequenos conjuntos de blocos.

Baixar Jogo (Pato)

Blocos Utilizados:

  • Quando Play
  • Quando Tocar
  • Mova
  • Pule
  • Toque o Som
  • Brilhe
  • Repita

O Ator Pato

Começaremos com o código que é iniciado pelo botão Quando Play, que significa que quando o jogo for iniciado o pato irá executar uma sequência de ações: ele se moverá para a direita utilizando o bloco Mova, emitirá um som utilizando o bloco Toque o Som moverá-se novamente, e então pulará três vezes usando o bloco Repita combinado com o Pule, depois ele se moverá mais uma vez para frente e emitirá novamente o som.

Também faremos um segundo separado, com a intenção de fazer o pato brilhar quando tocarmos nele, então utilizaremos o bloco Quando Tocar no pato, juntamente com o bloco Brilhe.
