Manual Octostudio

Nesta seção, você encontrará informações sobre o funcionamento de cada bloco do Octostudio. O objetivo é que você compreenda como cada parte contribui para a criação dos jogos e como usá-las de forma criativa. Se preferir, você também pode baixar o Manual Oficial do Octostudio, com todos os blocos descritos e exemplos de uso:

Baixar PDF

Blocos Amarelos - Quando Iniciar

Posicione um destes blocos no topo de cada script de código para definir quando iniciar.

Quando Play

Inicia o código quando o botão Play é tocado.

Quando Tocar

Inicia o código quando você toca no ator selecionado ou em outra opção.

Quando Mudar

Inicia o código quando o cenário selecionado aparece.

Quando na Borda

Inicia o código quando o ator atinge a borda selecionada da Tela.

Quando Encostar

Inicia o código quando o ator atual toca o ator selecionado.

Blocos Verdes - Movimento

Faça os atores se moverem.

Mova

Move o ator em uma direção e em uma velocidade específica.

Pule

Move o ator para cima e depois para baixo.

Gire

Gira o ator em um certo número de graus.

Vá Para

Move o ator para um ponto específico da tela.

Vá Para (Ator)

Move este ator para a posição de um outro ator.

Defina Direção

Define a direção para onde o ator esta apontado.

Avance

Faz o ator se mover na direção que ele está apontado.

Blocos Azuis - Palavras e Sons

Faça os atores tocarem um som, mostrarem um texto, e mais.

Toque o Som

Toca um som gravado ou da biblioteca.

Vibre

Faz o celular vibrar.

Diga

Exibe palavras em um balão de fala.

Blocos Roxos - Cenário

Faça alterações visuais, como redimensionar seus atores e alterar o cenário.

Defina Tamanho

Define o tamanho do ator para uma porcentagem do tamanho original.

Esconda

Faz o ator desaparecer da tela.

Mostre

Faz o ator aparecer na tela com a transparência desejada.

Mude Cenário

Troca o cenário atual para o cenário selecionado.

Organize

Posiciona um ator em frente ou atrás de outro ator.

Blocos Rosas - Cores e Luz

Mude a cor do seu ator e outras configurações de luz.

Brilhe

Cria um brilho ao redor do ator.

Blocos Laranjas - Controle

Controla o fluxo da sua programação.

Repita

Executa o código dentro deste bloco um número específico de vezes.

Sempre

Executa o código repetidamente até que o projeto pare.

Se (Condicional)

Verifica uma vez se a condição é verdadeira. Caso sim, executa os blocos de dentro.

Espere

Aguarda um número específico de segundos antes de continuar.

Pare

Para atores, sons em execução e até mesmo o jogo.

Transmita

Envia uma mensagem que todos os atores recebem.

Quando Receber

Inicia o código quando a mensagem selecionada é recebida.

Blocos Cianos - Mais Blocos

Inclui blocos diversos, como variáveis, ‘tilt’ e blocos personalizados.

Incline para Mover

Incline seu telefone para mover o ator.

Defina Variável

Define a variável selecionada para um número específico.

Mude Variável

Altera a variável selecionada por um determinado valor.

Mostre Variável

Mostra ou esconde a variável na tela.

Variável

Você pode encaixar este bloco em outro bloco para utilizar o valor atual da variável.

Informação

Informa o tamanho, posição e direção de um ator.

Aleatório

Escolhe um número aleatório dentro do intervalo especificado.

Operação

Executa uma determinada operação matemática em dois valores.
