Projetos do Curso Octostudio

Aqui você encontrará uma coletânea de projetos desenvolvidos com o Octostudio. Cada capítulo apresenta um jogo diferente com mecânicas únicas, pensadas para explorar os principais conceitos de programação visual.

Projeto 1

Neste projeto, vamos desenvolver um jogo em que o ator principal, a Flor, se movimenta para esquerda e para direita com ajuda de botões, este ator deve esquivar do fogo para não perder vida e coletar a água para marcar pontos.
